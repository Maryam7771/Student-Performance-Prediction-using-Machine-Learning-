# Student-Performance-Prediction-using-Machine-Learning-
Student performance evaluation is an important part of education. Manual 
assessment is time-consuming and may not always be accurate. Machine Learning 
provides an automated and efficient solution to analyze student data and predict 
outcomes. 
This system uses a Machine Learning model to predict student performance based 
on academic and behavioral features. A simple GUI built with Tkinter allows easy 
interaction for users.
